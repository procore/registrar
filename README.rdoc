= Registrar

Add `gem "registrar", github: "mcasper/registrar"` to your Gemfile

Copy migration: `rake registrar:install:migrations`
Run migration: `bundle exec rake db:migrate`

Add 'mount Registrar::Engine, at: "/sessions", as: "users"' to your `config/routes.rb` file
Add 'get "/auth/google/callback", to: "registrar/sessions#create"' to your `config/routes.rb' file

Start a rails server and visit localhost:3000/sessions to authorize!
